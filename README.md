# Single Page Application with Laravel 5.5 and Vue.js 2 - Invoice

### Frameworks and Libraries:

- Laravel 5.5
- Vue.js 2.5
- Vue Router
- Axios

### Provides:

- Progress bar
- Nested forms with selectable product
- CRUD



### Installation

`cd laravel-vue-js-spa-invoice`

`composer install`

`npm install`
